# Prompt Library

Prompts, skills, and workflows for building AI agent systems with [OpenClaw](https://github.com/openclaw/openclaw).

These are the actual prompts and configurations I use in my day-to-day setup — not theoretical, not hypothetical. If you've seen it in a video, the real version lives here.

## What's Inside

| Folder | What | Description |
|--------|------|-------------|
| [`defenseclaw/`](./defenseclaw/) | 🛡️ Security Setup | The ClawShield security layer prompt — scan every message for injection, PII, secrets, and more |
| [`agent-setup/`](./agent-setup/) | 🤖 Agent Identity | AGENTS.md, SOUL.md, and USER.md templates for giving your agent memory and personality |
| [`skills/`](./skills/) | 🧰 OpenClaw Skills | Drop-in skill files for specific capabilities |
| [`workflows/`](./workflows/) | 🔄 Workflows | Multi-step agent workflows (daily notes, content pipelines, task management) |

## How to Use

Most of these are designed for [OpenClaw](https://github.com/openclaw/openclaw), but the prompts and patterns work with any AI agent system. Copy what's useful, adapt what isn't.

For the full agent workflow setup (Obsidian + Linear + OpenClaw), see 👉 **[AI Agent Workflow](https://github.com/Jason-Cyr/ai-agent-workflow)**
