# Workflows

Multi-step agent workflows — things the agent does repeatedly that benefit from a structured prompt.

## Available Workflows

| Workflow | Description |
|----------|-------------|
| [`daily-briefing.md`](./daily-briefing.md) | Morning briefing — calendar, tasks, weather, carry-over items |
| [`end-of-day-reflection.md`](./end-of-day-reflection.md) | End-of-day reflection prompt for journaling |

## How to Use

These work as scheduled prompts (via OpenClaw cron) or as manual triggers. Each file contains the prompt and setup instructions.
